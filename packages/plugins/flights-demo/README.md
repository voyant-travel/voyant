# @voyant-travel/plugin-flights-demo

Demo `FlightConnectorAdapter` implementation for the standalone
`apps/flights-demo-api` service.

This package exists so OTA, tour-operator, and DMC templates can exercise
flight supplier integration flows without external credentials. It is a
reference adapter surface, not an airline or flight-operator implementation.

See [`apps/flights-demo-api`](../../../apps/flights-demo-api/README.md) for the
mock supplier service.
